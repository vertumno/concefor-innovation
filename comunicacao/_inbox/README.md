# _inbox — Caixa de entrada da CGTE

Jogue **tudo** que surgir aqui, sem se preocupar com organização: ideias soltas, arquivos,
prints, links, pedidos de outras coordenadorias, referências, rascunhos.

## Regras simples
- Não precisa nomear bonito. Se puder, prefixe com a data: `2026-07-15-ideia-reel-abertura.txt`.
- Um item por arquivo/pasta quando possível.
- Links soltos? Cole em `links-soltos.md` (crie se não existir).

## Processamento do inbox
Periodicamente (idealmente toda semana), processe cada item e mova para o destino:

| Se o item é… | Destino |
|---|---|
| Um fato novo do evento (data, palestrante, mudança) | `contexto/` (atualizar o doc certo) |
| Uma peça a produzir | `producao/<fase>/` + registrar no board |
| Uma tarefa / decisão de organização | `planejamento/board-producao.md` |
| Uma referência visual / inspiração | pasta da peça correspondente, em `_refs/` |
| Lixo / duplicado | apagar |

Depois de processar, **remova o item do inbox**. Inbox vazio = tudo distribuído.
