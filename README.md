# concefor-innovation

Cérebro único do **VIII CONCEFOR** — 17 a 20 de agosto de 2026, Cefor/IFES, Vitória-ES.
Tema dos 20 anos: *"Tecnologia, Transformação e Educação a Distância"*.

Fonte única de verdade do projeto, em duas áreas:

| Área | O que é | Entrada |
|---|---|---|
| **Inovação / App** | Contexto, decisões, spec e código do app do evento (PWA) | este README + [`CLAUDE.md`](CLAUDE.md) |
| **Comunicação (CGTE)** | Planejamento, produção e distribuição das peças de divulgação | [`comunicacao/README.md`](comunicacao/README.md) |

> (histórico git preservado) — ver [`decisoes.md`](decisoes.md).
