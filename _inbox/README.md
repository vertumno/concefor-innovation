# _inbox

Porta de entrada de arquivos brutos (transcrições, fotos, PDFs). Jogue aqui primeiro;
depois processe: mover pro lugar certo (`contexto/reunioes/`, `contexto/even3/`, …),
sintetizar em `.md` e registrar decisões extraídas em `../decisoes.md`.

**Inbox vazio é o estado bom.** Só este README mora aqui.
