# Produção — como funciona

Aqui ficam as peças de comunicação, organizadas por **fase**. Cada peça tem uma pasta própria.

## Fases
| Pasta | O que entra |
|---|---|
| [01-brindes-promocionais/](01-brindes-promocionais/) | Blocos, squeezes, canetas, camisas, crachá, bolo, sacola — tudo com selo 20 anos |
| [02-pre-evento/](02-pre-evento/) | Divulgação de palestrantes, eventos paralelos, contagem regressiva |
| [03-durante-evento/](03-durante-evento/) | Cobertura fotográfica, redes sociais ao vivo, podcasts, transmissão + Libras |
| [04-ideias/](04-ideias/) | Exploratório: podcasts de memória, app do Concefor |

## Anatomia de uma peça
Ao iniciar uma peça, crie uma pasta dentro da fase e coloque:
```
producao/02-pre-evento/card-palestrante-tessarolo/
├── brief.md            ← copie de templates/brief-peca.md e preencha
├── _refs/              ← referências, fotos-fonte, inspirações
└── (arquivos finais / links Canva no brief)
```

## Regras
- Toda peça nasce de um **brief** (ver [templates/brief-peca.md](../templates/brief-peca.md)).
- Todo status vive no **board** ([planejamento/board-producao.md](../planejamento/board-producao.md)) — a pasta guarda os arquivos; o board guarda o estado.
- Arte no Canva? Cole o link no brief da peça.
- Peça finalizada e publicada continua aqui (histórico); marque como ✅ no board.
