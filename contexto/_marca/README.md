# _marca — Assets de identidade visual

> Arquivos-fonte da marca e peças finalizadas que servem de **referência de padrão gráfico**
> para tudo que a CGTE produzir (cards de Instagram, stories, etc.).
> Documentação do padrão extraído: [../05-identidade-visual.md](../05-identidade-visual.md)

## Estrutura
```
_marca/
├── logos/
│   ├── concefor/        ← PNGs em alta do logo do Concefor
│   └── 20-anos-cefor/   ← assets do selo/logo 20 anos do Cefor
└── referencias-publicadas/
    ├── convite-autoridades/   ← arte finalizada do convite
    ├── post-instagram/        ← padrão de post (feed)
    ├── story-instagram/       ← padrão de story
    └── topo-formulario/       ← topo de formulário Google
```

## Regra de uso
- Estas peças finalizadas são o **padrão a seguir**: novos cards devem manter paleta,
  tipografia, posição de logos e assinatura coerentes com elas.
- Nunca editar os arquivos originais aqui — copiar para a pasta da peça nova e trabalhar lá.
